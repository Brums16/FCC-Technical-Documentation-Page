What is CSS?

CSS stands for cascading style sheets. It allows you to customise the appearance of an HTML document. It is used to set the properties of elements on a web page, such as the background colour, font size, margins and much more.

Although it would be possible to create a layout for a webpage using only html, CSS is a great way to ensure consistent design across a webpage. It also allows you to target multiple elements of the same type (more on this in the selectors section).

Prerequisite Knowledge

Since CSS is a style sheet programming language, the most important thing to know already is how to create an HTML document. Once you know how to create headings, paragraphs and divisions in your HTML document, you can begin using CSS to improve the appearance of your HTML page. You'll also want a basic understanding of how to use <href> in order to link your fancy new stylsheet to your html page.

Similar to learning any new language, the key to mastering CSS is consistent practice and applying your skills. It's a good idea to have some plan as to what you want to learn or build so you can make sure you're on the right track.

Getting Started

The first thing you'll need to do is link your html code to your CSS code. This is pretty straightforward, just use an href like you would to link to a webpage, but instead you link it to to your stylesheet. Usually you'll place this link in the head section of your page. It's also good practice to describe the relationship to the .css file, using the rel="stylesheet" syntax.

So in bullet point form to satisfy 'the man' at freeCodeCamp:

  • First, create the html for your page (depending on your layout you might want to start working on your css alongside it!)
  • Link your html file to your CSS file. Don't forget to describe the relationship using rel syntax.
  • Use CSS to style your fonts, layouts, colours and more. This is typically done using a selector of some kind.

Once you've selected the part of your code you wish to format, for example your h1 text, you can start to format it using CSS. Let's say you wanted your h1 to be a bit larger with white text on a blue background, your CSS would look like this.

h1 {
font-size: 2.5rem;
background-color: blue;
color: white
}

Font Styling

CSS allows you to make changes to font styles across your webpage. Here are a few things you can change:

  • The font color, by using color
  • The font size, using (shocker) font-size
  • The font family, using font-family. It's good practice to include back-up fonts too, as follows:
h1 {
font-size: 2.5rem;
font-family: Helvetica, sans-serif}

It's important to include back-ups like this when writing CSS in case the browser or device the user has displays the content slightly differently.

The Box-Model

CSS wraps a box around every element in your HTML. Here's a great image to illustrate how the box-model works:

There are 4 sections to the box-model, going from outside to inside:

  1. Margin: clears a transparent area outside the border.
  2. Border: inside the margin, around the padding and the content. The border can be given a particular colour, weight and style using CSS.
  3. Padding: A transparent area around your content inside the border. Great for making text stand out!
  4. Content: Your exciting content containing text or images.

Adding Colour

CSS allows you to add a range of colour styles to your web page. Even though it was created by a Norwegian, an Englishman, and a Belgian you need to spell colour wrong everytime you use it.

CSS allows you to define a colour in three ways:

RGB rgb(89,149,219)
HEX#5995db
HSL212,64,60

Each of these codes mean the same shade of blue in the table. RGB and HEX are the most commonly used.

Selectors

Selectors tell your CSS where to work its magic. You can use a simple selector to select elements based on their name, id or class:

Type Example Selector
Element<h2>h2
Class <div class="big-box"> .big-box
ID <div id="client"> #client
